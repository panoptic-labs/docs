 An ERC-1155 ID representing a position. 


```graphql
type TokenId {
  id: ID!
  idHexString: String!
  pool: Pool!
  tokenCount: BigInt!
  accountBalances(
    skip: Int = 0
    first: Int = 100
    orderBy: AccountBalance_orderBy
    orderDirection: OrderDirection
    where: AccountBalance_filter
  ): [AccountBalance!]!
  legs(
    skip: Int = 0
    first: Int = 100
    orderBy: Leg_orderBy
    orderDirection: OrderDirection
    where: Leg_filter
  ): [Leg!]!
}
```
