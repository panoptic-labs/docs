 A chunk of liquidity. Used to track liquidity available in Uniswap, as well as liquidity removed from Uniswap for use in PanopticPools. 


```graphql
type Chunk {
  id: ID!
  owner: Account!
  manager: Account!
  tickLower: BigInt!
  tickUpper: BigInt!
  strike: BigInt!
  width: BigInt!
  tokenType: BigInt!
  pool: Pool!
  panopticPool: PanopticPool
  netLiquidity: BigInt!
  shortLiquidity: BigInt!
  longLiquidity: BigInt!
  shortCounts: BigInt!
  longCounts: BigInt!
  totalLiquidity: BigInt!
  legs(
    skip: Int = 0
    first: Int = 100
    orderBy: Leg_orderBy
    orderDirection: OrderDirection
    where: Leg_filter
  ): [Leg!]!
}
```
