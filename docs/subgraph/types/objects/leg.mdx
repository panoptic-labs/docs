 A leg of a TokenId, representing a single option. 


```graphql
type Leg {
  id: ID!
  index: Int!
  idHexString: String!
  pool: Pool!
  optionRatio: BigInt!
  asset: BigInt!
  tokenType: BigInt!
  isLong: BigInt!
  riskPartner: BigInt!
  strike: BigInt!
  width: BigInt!
  chunk: Chunk!
  legCount: BigInt!
  tokenIds(
    skip: Int = 0
    first: Int = 100
    orderBy: TokenId_orderBy
    orderDirection: OrderDirection
    where: TokenId_filter
  ): [TokenId!]!
}
```
